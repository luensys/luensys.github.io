OSX에 brew설치하기

brew에 대해 아는 것이 좀 적다보니 brew는 사용하지 않으려 했으나,
php 설치할 때 기존에 사용했던 방식(https://php-osx.liip.ch/)을 이용해 설치를 하면
추가적으로 손 볼 것이 생기게 되어 결국 전체적으로 brew를 이용하기로 했다.

설치하는 방법은 BREW 홈페이지에 자세하게 설명이 되어 있다.

참조사항
설치할 때 관리자 권한이 필요하나, 권한 없이 설치를 하게 되어도 관리자 비밀번호를 요구하게 된다.
